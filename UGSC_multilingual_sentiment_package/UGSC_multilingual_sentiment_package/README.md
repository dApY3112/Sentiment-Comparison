# UGSC Multilingual Sentiment Analysis Dataset

This repository contains the multilingual version of the User Gold Standard Corpus (UGSC) used for zero-shot sentiment classification in the domain of sustainable urban mobility.

## Contents

- `translated_versions/`: Human-aligned translations of the original English corpus into Spanish, French, German, and Italian.
- `sentiment_predictions/`: Sentiment classification results using XLM-RoBERTa (zero-shot).
- `scripts/`: Placeholder folder for inference and analysis scripts (to be added).

## Citation

If you use this dataset or findings from the associated publication, please cite:
